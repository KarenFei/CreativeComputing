upload homework to Github

before you begin:

  • find your github folder
  • all your new files need to be in this folder

Step 1

in terminal type cd  "cd space", drag folder, hit "return".

Step 2

add any new files (replace "newpage.html" with your file)

git add newpage.html

If your file is in one or more folders it looks like this:

git add homework/01/newpage.html

a quick way to add ALL files is:

git add homework/01/

this adds all the files inside of the folder "01" inside "homework"

Step 3

commit and push

git commit -am "a message to yourself, describe what changed/what is new"

all your work, changes, new files are now saved, but only on your computer.

git push

and if git push doesn't work try git push origin master

now they are on the internet. You're done!
